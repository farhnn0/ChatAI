# Copywriting Guidelines

## Tone

Use short and practical text. The app should sound like a developer tool, not a marketing landing page.

## Good UI Text

```txt
New chat
Search chats
Settings
Message your AI...
Thinking...
Generating response...
Local model
Cloud model
Clear chat
```

## Avoid

```txt
Unleash your productivity
Revolutionize your workflow
Supercharge your coding journey
Your ultimate AI companion
```

## Empty State Copy

Option 1:

```txt
What are you working on?
Choose a model and start a conversation.
```

Option 2:

```txt
Start a new conversation
Use local or cloud models from one workspace.
```

Option 3:

```txt
Ask anything about code, debugging, or your project.
```

## Error Copy

Ollama offline:

```txt
Ollama is not reachable. Make sure it is running on localhost:11434.
```

DeepSeek API error:

```txt
DeepSeek request failed. Check your API key or account balance.
```

No model selected:

```txt
Select a model before sending a message.
```

Empty prompt:

```txt
Type a message first.
```
