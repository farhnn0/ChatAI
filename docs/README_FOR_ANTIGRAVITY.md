# Read This First

This documentation describes a Next.js local AI chat app that combines Ollama and DeepSeek in one clean interface.

Before coding, read these files in order:

1. `README.md`
2. `TECH_STACK.md`
3. `DESIGN_SYSTEM.md`
4. `AI_MODELS.md`
5. `ARCHITECTURE.md`
6. `API_CONTRACT.md`
7. `PROJECT_STRUCTURE.md`
8. `IMPLEMENTATION_PLAN.md`
9. `ANTIGRAVITY_INSTRUCTIONS.md`
10. `ENVIRONMENT.md`
11. `COPYWRITING.md`

Main instruction:

Build the UI first. Do not overbuild backend features before the interface is stable.

Design must be dark only, no emoji, no gradient, and clean enough to feel like a serious developer workspace.
